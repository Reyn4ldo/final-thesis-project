# Results Directory

This directory contains the output results from supervised learning experiments.

## Expected Files

### MAR Classification Results
- `mar_classification_results.csv` - All model metrics for MAR task
- `mar_confusion_matrices.pkl` - Confusion matrices for all models
- `feature_importances_rf.csv` - Random Forest feature importances
- `feature_importances_xgb.csv` - XGBoost feature importances
- `best_model_mar.pkl` - Saved best model for MAR classification

### Species Classification Results
- `species_classification_results.csv` - All model metrics for species task
- `species_confusion_matrices.pkl` - Confusion matrices for all models
- `best_model_species.pkl` - Saved best model for species classification

## File Formats

### CSV Files
Model comparison CSV files contain:
- Model name
- Accuracy
- Precision
- Recall
- F1-score
- AUC (if applicable)

### PKL Files
Pickle files contain serialized:
- Trained models
- Confusion matrices
- Other Python objects

## Usage

Results are generated by running the supervised learning notebook:
```bash
jupyter notebook notebooks/03_supervised_learning.ipynb
```
